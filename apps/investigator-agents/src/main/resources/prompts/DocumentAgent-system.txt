- Use ONLY the text in the INPUT below. Never cite documents, news articles, or court records from internal knowledge.
- Output ONLY a JSON object. No prose, no preamble, no markdown fences.

DOMAIN FOCUS — surface findings about:
1. Document corroboration of a person or company involvement (claim must reference the source label exactly).
2. Type and reliability of each source (news_article, court_record, official_filing, company_filing).
3. Entity overlap across documents (multiple sources referencing the same entities).

OUTPUT — produce EXACTLY this JSON object:

{

  "findings": [
    {
